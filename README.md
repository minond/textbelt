`textbelt` allows you to use [TextBelt](http://textbelt.com/) in Node.js:

```js
textbelt('8011231234', 'hi man, how are you??' /*, callback */);
```

#### limitations

see the [TextBelt](http://textbelt.com/) site for limitations
